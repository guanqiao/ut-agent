## 目标

将测试覆盖率从 50% 提升到 65%+

## 计划添加的测试

### 1. selection 模块测试 (约 620 语句)

* **test\_selection\_change\_detector.py** - 测试代码变更检测

* **test\_selection\_impact\_analyzer.py** - 测试影响分析

* **test\_selection\_priority.py** - 测试优先级计算

* **test\_selection\_test\_selector.py** - 测试用例选择

### 2. tools 模块补充测试

* **test\_tools\_mutation\_analyzer.py** - 测试变异分析

* **test\_tools\_change\_detector.py** - 补充变更检测测试

### 3. memory 模块补充测试

* **test\_memory\_long\_term.py** - 长期记忆存储

* **test\_memory\_short\_term.py** - 短期记忆存储

* **test\_memory\_manager.py** - 记忆管理器

* **test\_memory\_semantic.py** - 语义记忆

* **test\_memory\_preference.py** - 用户偏好

### 4. graph 模块补充测试

* **test\_graph\_graph.py** - LangGraph 工作流构建

### 5. ci 模块补充测试

* **test\_ci\_runner.py** - CI 运行器

## 预期结果

* 新增测试约 200-250 个

* 覆盖率提升至 65%+

* 所有 selection 模块达到 60%+ 覆盖率

* tools/mutation\_analyzer 达到 50%+ 覆盖率
